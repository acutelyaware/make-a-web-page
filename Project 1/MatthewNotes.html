Important Concepts

Lesson 1: Being Basic

What is The World Wide Web?

The World Wide Web is comprised of a collection of HTML documents. When a person accesses a website through a web browser, the computer sends the HTTP request through a server and retrieves the correct HTML document, which the browser is then able to interpret and display. This video is a good explanation

HTML

HTML stands for Hypertext Markup Language. It is the language predominantly used for content on the web. It consists of text content (what you see) and markup(what it looks like). The video here is a good refresher.

Elements and Tags

HTML markup consists of tags. Most tags have an opening and closing tag, with the content affected inbetween the tags, however some only need the opening tag. These are known as void tags. Tags and content together are known as elements. HTML documents consist of many elements. Here is a video that explains more.

Inline & Block Elements

Elements are either inline or block. Block elements form an invsible box around the content, while inline just forces a line break. So far inline tags have been mostly void tags, while block elements are not.

Lesson 2: Creating a Structured Document

Creating a Structured Document

Web pages are described as having a tree-like structure. This refers to the way that elements can contain other elements, like Russian nesting dolls. In the browser these elements show up as boxes nested inside of each other. Every style element is a box.

In Chrome you can use developer tools to view the code for a page. The tree-like structure can be observed as elements appear as sideways triangles that can be expanded and further explored. Additionally you can change settings in developer tools to see how it would affect how the code is interpretted.

Text Editing Programs

While HTML can be written in a plain text document, specific programs called text editors make it simpler and more efficient to insert opening/closing tags, and inserting other coding needs. There are different types of programs that serve different uses. For example Scratchpad shows changes to code immediately, but is more suited for small projects. Sublime Text is much more powerful, but does not show results instantaneously.

Lesson 3: Stylin' & Profilin'

CSS

CSS is used to control and change the style of HTML elements. CSS stands for Cascading Style Sheets. The cascading part refers to the fact that the most specific rule applies to all elements. For example, a style applied to a larger element would also be applied to all of the elements nested within that larger element, unless otherwise specified. This is known as the concept of inheritance.

CSS styles can be applied to a specific tag, for example to a bold tag, making all of the elements in the HTML with that tag be affected, or it can be applied to an entire class. This can be achieved by giving entire elements a class name, and then applying a specific style to that class

Don't Make Me Repeat Myself

It is important to avoid reptition whenever possible. Reptition provides a greater chance for errors, and even 1 small error can affect the results of your code.

Consistency throughout an HTML document allows for styles to be applied more broadly without reptition. For example in this document each lesson was broken into a div and given the same class name. This allows each lesson to be given the same style without having to repeat the style for each lesson, and allows to change the overall look more quickly.
